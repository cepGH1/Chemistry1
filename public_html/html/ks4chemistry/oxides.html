Oxides

Any compound with any single element and oxygen in it.

Often produced by direct reaction of the element with oxygen.
For metals this process is called corrosion. The oxides produced are much weaker than the metals and so this is often an unwelcome process.

Acidic and Basic Oxides

Basic

Metals mostly produce basic oxides.
Some produce amphoteric(neutral) oxides
Example 1

Magnesium + Oxygen --> Magnesium oxide

2Mg + O2 --> 2MgO

Example 2

Copper + Oxygen --> Copper oxide

2Cu + O2 --> 2CuO

Acidic

Non-metals produce acidic oxides.

Example 1

Sulfur + oxygen --> sulfur dioxide

S + O2 --> SO2

Sulfur dioxide dissolves in water to give an acidic (low pH) solution
This is one of the major things in acid rain
Example 2

When the temperature is very high, nitrogen (impurities in fuels) can react with oxygen to make nitrogen oxides. These compounds are air pollutants.

Nitrogen + Oxygen --> nitrogen monoxide

N2 + O2 --> 2NO

Nitrogen monoxide can react with more oxygen to produce nitrogen dioxide

Nitrogen monoxide + Oxygen --> Nitrogen dioxide

2NO + O2 --> 2NO2
