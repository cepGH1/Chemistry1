Reversible Reactions

In these reactions the products can react together to reproduce the reactants

3H2(g) + N2(g) → 2NH3(g)
NH3(g) → N2(g) + 3H2(g)

These reactions work from left to right and from right to left
We write this type of reaction with a special symbol instead of an arrow
⇌
3H2(g) + N2(g) ⇌ 2NH3(g)
