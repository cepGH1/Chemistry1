Displayed Formula

Used to show the way atoms are connected in molecules

Formula showing covalent bonds as lines between element symbols
Single line for a single covalent bond
Double line for a double covalent bond
Triple line for a triple covalent bond

Examples

Ethyne C2H2

Ethyne has one triple bond between the carbons and each hydrogen is bonded to a carbon by a single bond

Ethene C2H4

Ethene has one double bond between the carbons and each hydrogen is bonded to a carbon by a single bond

Ethane C2H6

Ethane has only single bonds
