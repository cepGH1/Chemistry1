Acids and their Reactions

Acid means sour (from the latin word acidus)
Some foods contain acids and we recognise these can be sour.
Lemons contain citric acid.
Vinegar is ethanoic acid dissolved in water.
These are all weak acids that we see every day.

Reactions with metals

Reactive metals (metals above hydrogen in the reactivity series) react with acids to make a metal salt and hydrogen gas.

Example

Magnesium + Hydrochloric Acid → Magnesium Chloride + Hydrogen

Mg(s) + 2HCl(aq) → MgCl2(aq) + H2(g)

Metals less reactive than hydrogen do not react with dilute acids

Reactions with bases ( Neutralisation Reactions)

Acid + Base → Salt + Water

Examples

Hydrochloric Acid + Sodium Hydroxide → Sodium Chloride + Water
HCl(aq) + NaOH(aq) → NaCl(aq) + H2O(l)

Sulfuric Acid + Copper Oxide → Copper Sulfate + Water
H2SO4(aq) + CuO(s) → CuSO4(aq) + H2O(l)

Reactions with hydrogencarbonates

Acid + HydrogenCarbonate → Salt + Carbon Dioxide + water

Example
Hydrochloric Acid + Sodium Hydrogencarbonate → Sodium Chloride + Carbon Dioxide + Water
HCl(aq) + NaHCO3(s) → NaCl(aq) + CO2(g) + H2O(l)
