Esters

An ester is an organic compound that has an ester linkage(functional group) in it.

For example

Making esters

Esters can be made using condensation reactions from carboxylic acids and an alcohols

CH3COOH + CH3OH --> CH3COOCH3 + H2O

Naming Esters

They are named for the acid starting material.
In practice name them from right to left, with 'noate' on the end
The example above made from methanol and ethanoic acid is methyl ethanoate
