Ethene

The simplest alkene. Because it contains a double covalent bond it is unsaturated.
The double bond allows chemical reactions to take place quite easily
Addition across the double bond and polymerisation

Addition across the double bond

1. Bromine (other halogens react similarly)

ethene + bromine --> 1,2-dibromoethane

C2H4 + Br2 --> C2H4Br2

The bromine in this reaction can be from bromine water. This is just bromine dissolved in water. The colour of the bromine disappears when it reacts. This can be used as a test to see if an organic compound contains a double covalent bond between 2 carbon atoms.

2. Hydrogen Bromide (other hydrogen halides react similarly)

ethene + hydrogen bromide --> bromoethane

C2H4 + HBr --> C2H5Br

Polymerisation

ethene --> polyethene

nC2H4 --> -[C2H4]n -

String of beads model

This can be thought of as making a chain of beads. The polymer is the chain of beads. Each bead is made from one ethene molecule.
