How to name some simple organic molecules

  1. Count how many carbons are joined one after another (like a string of beads)- This is the carbon backbone of the molecule

    1 single carbon = meth
    2 carbons = eth
    3 carbons = prop
    4 carbons = but
    5 carbons = pent
    6 carbons = hex

    This makes the group name of the compound

  2. Look for other groups in the molecule

    only hydrogen atoms, it's an alkane: name ending : yl
    OH group, it's an alcohol: name ending : anol
    double bond between carbons, it's an alkene: name ending : ene
    triple bond between carbons, it's an alkyne: name ending : yne
    COOH group, it's a carboxylic acid: name ending : anoic acid

  3. Remember to name extra methyl (or other alkyl) groups attached to the carbon backbone
