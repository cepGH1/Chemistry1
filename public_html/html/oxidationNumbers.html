Oxidation Numbers

How to work them out

There are a few rules to remember in order to do this

1. Neutral elements have oxidation number 0 (Neutral:None)
2. the oxidation number of a monatomic ion is the same as it's charge
eg. Na+ = +1, and Mg2+ = +2
3. The sum of the Oxidation Numbers in a thing is always equal to it's charge
ie. if it is overall neutral then the oxidation numbers add up to 0
4. Group 1 and 2 elements have oxidation numbers +1 and +2 respectively
5. O has oxidation number -2 unless it is bonded to F
6. F always has oxidation number -1 in it's compounds
